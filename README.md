# qtd-vuejs

A simple quote of the day app that connect to my local socket

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and run socket server
```
python3 server.py
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
